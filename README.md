# Phishing Website Detection using Machine Learning 

## Project Description

This project reproduces and evaluates a published machine learning solution for phishing website detection.

The goal is to classify websites as either phishing or legitimate using features extracted from URLs and website characteristics.

As part of this project, I performed exploratory data analysis (EDA), trained and compared multiple machine learning models, evaluated their performance, and critically analyzed the claims made in the original project.

The implemented models are:

- Logistic Regression
- Decision Tree
- Random Forest

The models were evaluated using Accuracy, Precision, Recall, F1-score, and Confusion Matrix.

## Execution Instructions

1. Clone or download this repository.
2. Make sure that the dataset file (`5.urldata.csv`) is located in the same folder as the notebook.
3. Install the required Python libraries:

- pandas
- numpy
- matplotlib
- seaborn
- scikit-learn

4. Open the Jupyter Notebook.
5. Run all notebook cells from top to bottom.

## Dataset Source

The dataset used in this project contains 10,000 websites:

- 5,000 phishing websites collected from PhishTank.
- 5,000 legitimate websites collected from the University of New Brunswick (UNB).

The dataset included in this repository is:

`5.urldata.csv`

Original sources:

PhishTank:
https://www.phishtank.com/

University of New Brunswick (UNB):
https://www.unb.ca/cic/datasets/url-2016.html
